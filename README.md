# NextJS Template Repo
###### By Daniel WIlder

---

This repository is a template repo for nextjs projects. 
Features Include:
- NextJS app router
- Typescript
- TailwindCSS
- Shadcn UI
- husky precommit
- linting
- formatting
- typed env
---

#### Install

```bash
yarn
```

#### Run

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
